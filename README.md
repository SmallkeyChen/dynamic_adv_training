# Dynamic AdveRsarial Training (Dynamic/DART) 
Code for ICML2019 Paper "On the Convergence and Robustness of Adversarial Training"

## Requirements
- Python 3.5.2, 
- Tensorflow 1.10.1 
- Keras 2.2.2

## Usage

Simply run the code by: python3 train_models.py
